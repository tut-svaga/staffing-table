# Staffing Table (C++ консольное приложение)

Это мой консольный проект на C++, который реализует систему управления сотрудниками (штатное расписание).

Я делал его, чтобы прокачать:
- ООП
- работу с STL
- работу с файлами
- алгоритмы сортировки
- правильную структуру проекта из нескольких файлов

---

## Возможности программы

✔ Добавление сотрудников  
✔ Редактирование данных  
✔ Удаление сотрудников  
✔ Поиск по:
- имени
- фамилии
- должности
- ID
- зарплате  

✔ Сортировка:
- Quick Sort
- Shell Sort
- Bubble Sort  

✔ Сохранение одного или всех сотрудников в файл  
✔ Загрузка данных из файла  

---

## Используемые технологии

- C++17
- STL (vector, string, functional)
- fstream
- Ручная реализация алгоритмов сортировки
- Многофайловая архитектура
